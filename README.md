# Rest-API

Node js, PostgreSQL, Javascript.

## API Endpoints

1. GET /students
   `localhost:3000/students`
2. GET /students/:id
   `localhost:3000/students/1`
3. POST /students
   `localhost:3000/students`
4. PUT /students/:id
   `localhost:3000/students/1`
5. DELETE /students/:id
   `localhost:3000/students/1`

## Database

### Students Table

1. name
2. email
3. age
4. dob (date of birth)
